# Аудит анимаций (improve-animations, standard) — коммит 3536225

Методика: `.claude/skills/improve-animations` (Emil Kowalski). Область: публичный
редизайн (`.rd` в `app/globals.css`, `components/home/*`, шапка, модалки
конструктора). Личность продукта: премиальный спокойный спа — движение должно
быть плавным, но отклик мгновенным.

## Разведка

- Стек: без motion-библиотек; чистый CSS (`transition`, 2 `@keyframes`),
  инлайн-grid-rows аккордеон, IntersectionObserver для `.reveal`.
- Токенов easing/duration НЕТ: всюду рукописные `0.25s`, `0.3s ease`,
  `0.7s ease`, `1.4s ease` со встроенными слабыми кривыми.
- reduced-motion: глобальный «глушитель» 0.01ms на всё (globals.css:263).
- Частотная карта: хверы кнопок/карточек — десятки раз за визит; модалки
  согласия — каждый заказ (критический флоу); бургер — каждый мобильный визит.

## Дополнительная находка (в ходе применения)

- **`.reveal` был мёртвым кодом**: CSS и `RevealInit` существовали, но ни один
  элемент страницы класса не имел — эффект появления секций из утверждённого
  макета вообще не работал. Исправлено: `RevealInit` смонтирован на главной,
  `reveal` навешан на wrap'ы секций (о нас, атмосфера, программы, сертификаты,
  салоны, перед визитом, советы).

## Упущенные возможности (2)

- Подпись слайда hero (`.show-cap`) меняется мгновенным свапом текста —
  просится микро-fade 200ms.
- Финальный экран «Заказ создан» (`builder-client`) — редкий эмоциональный
  момент вообще без делайта.
